# Notifications

Channels: email (mandatory), in-app, optional SMS/push.

Events:
- Application approved/rejected
- Submission verified/rejected
- Milestones: 1k/10k/100k views; nearing cap; payout initiated/completed
- Budget nearing exhaustion; campaign paused/completed

Delivery:
- Use provider (e.g., Postmark, Sendgrid)
- Template versioning and localization
- User preferences and rate limits
