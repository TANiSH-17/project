# Roadmap

## MVP (Weeks 0–6)
- User auth (OAuth), basic profiles
- Brand creation and campaign publishing (CPM, budget, start/end)
- Campaign browse and detail page; applications and approvals
- Per-participant tracking links; basic click tracking
- Manual content submission and verification workflow
- Earnings accrual (views/clicks → pending earnings)
- Admin panel basics; manual payout recording (no real money yet)

## V1 (Weeks 6–12)
- Stripe Connect onboarding + KYC; automated payouts (sandbox → prod)
- Social API integrations: YouTube (view counts), TikTok, Instagram where feasible
- Automated metric ingestion workers + holds/locking
- Ledger + reports; campaign-level and user-level analytics
- Anti-fraud v1: velocity limits, denylist, duplicate detection
- Email and in-app notifications

## V2 (Weeks 12–20)
- Advanced targeting and allowlist rules; self-serve review templates
- Conversion tracking (pixels, postbacks), multi-touch attribution options
- Budget pacing with hour/day throttles; smart acceptance
- Brand webhooks and API keys; external developer docs
- Risk scoring model and review tooling; disputes workflow

## Nice-to-haves
- Mobile app or PWA enhancements
- Multi-currency with FX
- Referral program and bonuses
- A/B testing of briefs and payouts
