# Feature Set

## For Brands/Large Creators
- Organization accounts, team members, RBAC
- Campaign creation with budget, CPM/CPA rules, caps, start/end, targeting (geo, platform, language)
- Creative brief builder: assets, examples, hashtags, do/don'ts, disclosure guidelines
- Application management: allow/deny lists, auto-approval rules, screening questions
- Real-time performance dashboard: spend, CPM effective, top participants, anomalies
- Payout controls: holds, batch payout approval, clawbacks
- Webhooks + API keys for programmatic control

## For Participants (Creators/Fans)
- Profile with linked social accounts and verification
- Campaigns browse/search/filter (platform, payout rate, difficulty)
- Application & acceptance flow; quotas and per-user caps visibility
- Tracking link generator; content submission and status tracking
- Earnings dashboard with pending/locked/paid; payout preferences
- Notifications: approvals, milestones, payouts, issues

## Admin/Operations
- Review queues for applications, submissions, fraud signals
- Dispute management and resolution flows
- Financial reconciliation: ledger views, payout batches, chargebacks
- System health dashboards; alerting on anomalies

## Integrations
- Stripe Connect + Identity, YouTube/TikTok/Instagram/X APIs
- Email provider, SMS (optional), push (PWA)
- Link shortener/redirect service, analytics pipelines
